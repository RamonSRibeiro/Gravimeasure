Política de Privacidade

1. Introdução

A Rede Gravimétrica Fundamental opera o sistema GraviMeasure. Esta página informa você sobre nossas políticas e práticas de privacidade relativas à coleta, uso e divulgação de dados pessoais.

2. Coleta de Dados

Coletamos dados pessoais que você voluntariamente nos fornece, incluindo:

  • Nome e e-mail durante o registro
  • Informações de perfil de usuário
  • Dados de medições gravimétricas que você insere no sistema
  • Histórico de acesso ao sistema

3. Uso de Dados

Utilizamos os dados coletados para:

  • Gerenciar sua conta e fornecer acesso ao sistema
  • Armazenar e organizar dados de medições gravimétricas
  • Melhorar e otimizar nossos serviços
  • Comunicar-nos com você sobre atualizações do sistema
  • Cumprir obrigações legais e regulamentares

4. Segurança de Dados

Implementamos medidas técnicas e organizacionais apropriadas para proteger seus dados pessoais contra acesso não autorizado, alteração, divulgação ou destruição.

5. Retenção de Dados

Retenemos seus dados pessoais enquanto sua conta estiver ativa. Se você desativar ou solicitar a exclusão de sua conta, retiraremos seus dados pessoais de acordo com nossas políticas de retenção legais.

6. Compartilhamento de Dados

Não compartilhamos seus dados pessoais com terceiros sem seu consentimento explícito, exceto quando obrigado por lei ou quando necessário para fornecer nossos serviços.

7. Seus Direitos

Você tem o direito de:

  • Acessar os dados pessoais que mantemos sobre você
  • Corrigir dados pessoais imprecisos
  • Solicitar a exclusão de seus dados pessoais
  • Revogar seu consentimento para o processamento de dados

8. Alterações a Esta Política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos você sobre quaisquer mudanças materiais por meio de um aviso destacado neste site ou por email.

9. Contato

Se você tiver dúvidas sobre esta Política de Privacidade ou nossas práticas de privacidade, entre em contato conosco através do formulário de contato no sistema.
